# Опис

Телеграм бот, який виконує функції асистента.

## Функції

- експорт тексту з PDF файлів (PyMuPDF)
  Поведінка: користувач надсилає PDF → бот завантажує файл, витягує текст і готує текстовий файл і віддає користувачу
- після експорту отриманий файл виконується адаптації тексту:

✔️ прибрати сухі технічні фрагменти
✔️ розбити на логічні блоки
✔️ переписати складні речення простіше
✔️ додати уточнення / пояснювальні фрази
✔️ форматувати списки в текстовий опис
✔️ підготувати заголовки / паузи / проміжні вступи
✔️ усунути дублікати / нерелевантні фрагменти

- результат кожного воркеру відправляється користувачу в чат

- під час роботи воркерів користувач може переглядати лог роботи воркера в чаті
